CSS boje

Boje možemo zadavati:

  • Imenima

    ova boja je 
    blue
  • Kombinacijom crvene, zelene i plave

    • RGB format: 
       rgb(jacina_crvene, jacina_zelene, jacina_plave)
      ova boja je 
       rgb(255, 165, 0)
    • RGBA format: 
       rgba(jacina_crvene, jacina_zelene, jacina_plave, jacina_providnosti)
      ova boja je 
       rgba(255, 165, 0, 0.5)
    • HEX format: 
       #CCZZPP
      CC - broj u heksadekadnom formatu koji definiše jačinu crvene
      ZZ - broj u heksadekadnom formatu koji definiše jačinu zelene
      PP - broj u heksadekadnom formatu koji definiše jačinu plave

      ova boja je 
       #ff0000
  • Kombinacijom nijanse, zasićenja i svetlosti

    • HSA format: 
       hsa(nijasna, zasicenje, svetlost)
      ova boja je 
       hsl(120, 100%, 25%)
    • HSLA format: 
       hsla(nijasna, zasicenje, svetlost, jacina_providnosti)
      ova boja je 
       hsla(120, 100%, 25%, 0.5)
